# Giapha

Giapha is a local-first family tree and family-state tracker inspired by common genealogy and genogram workflows.

## Features

- Visual family tree with parent-child and partner links
- Gender-colored person nodes with wellbeing state markers
- Zoom controls for big-picture tree review
- Search and filters by life status and wellbeing
- Editable profile state for each person
- Add parent, partner, or child from the selected profile
- Dated state timeline with confidence and location
- Source ledger for records, interviews, documents, photos, web links, and archive paths
- Memory links for stories, photos, documents, audio, video, and other artifacts
- Problem spotter for missing dates, stale state checks, unsourced events, and age anomalies
- Private JSON backup/export and redacted public export
- LocalStorage persistence with no backend required
